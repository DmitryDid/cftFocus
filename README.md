# cftFocus
Консольное приложение сортировки слиянием нескольких файлов в один.
Возможна сортировка строковых/числовых данных(целые числа).
Предусловие: данные в файлах предварительно отсортированны.
Технология реализации: JAVA 8

Параметры программы задаются при запуске через аргументы командной строки:
1. режим сортировки (-a или -d), необязательный, по умолчанию сортируем по возрастанию;
2. тип данных (-s или -i), обязательный;
3. имя выходного файла, обязательное;
4. остальные параметры – имена входных файлов, не менее одного.

Примеры запуска из командной строки для Windows:
sort-it.exe -i -a out.txt in.txt (для целых чисел по возрастанию)
sort-it.exe -s out.txt in1.txt in2.txt in3.txt (для строк по возрастанию)
sort-it.exe -d -s out.txt in1.txt in2.txt (для строк по убыванию)
Путь к файлу можно педавать как абсолютный, так и относительный.

Особенности текущей реализации:
- не обрабатываются файлы не помещающиеся в оперативную память. (в разработке)
- выполняется проверка обязательных параметров.
- параметры можно подавать в любом порядке.
- перед сортировкой чисел выполняется очистка данных - отбрасываются данные, которые не удается распарсить.
- если строка содержит пробельные символы между корректными данными, то все данные учавствуют в сортировке.
- обрабатываюся все исклютительные ситуации, с информированием пользователя.
- написаны тесты.
